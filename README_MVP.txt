🎮 CONTROLES DO MVP:

WASD            → Mover
Mouse           → Olhar ao redor

DEBUG KEYS:
F1              → Ver estado (fragmentos coletados, etc)
F2              → Forçar Final Bom
F3              → Forçar Final Ruim

    → Maria é aprisionada → FINAL BOM ✅

OU

Jogador spawna → Sai sem fragmentos
    → Tenta escapar → Maria não foi aprisionada
    → FINAL RUIM ❌

⚡ DICAS IMPORTANTES:

1. Certifique-se de criar a tag "Player" no projeto
2. Sempre faça os campos "public" ou [SerializeField] assignáveisno Inspector
3. Use o Console (Ctrl+Shift+C ou Window → General → Console)
4. Os scripts usam GameManager.Instance em vários lugares
5. Os fragmentos precisam de IDs únicos (1, 2, 3)
6. O ritual só funciona com os 3 fragmentos
7. A câmera fica trancada no início (pressione ESC para debug)


═══════════════════════════════════════════════════════════════════

❓ PERGUNTAS FREQUENTES:

P: Posso mudar nomes dos GameObjects?
R: Sim, desde que não seja "Player" (ele tem tag)

P: Como adicionar som?
R: Crie GameObjects com AudioSource e chame Play() nos scripts

P: Preciso de modelos 3D?
R: Não! Cubes e Spheres funcionam para MVP

P: Como testar os finais?
R: Use F2 e F3 ou faça o caminho completo no jogo

P: Posso adicionar mais fragmentos?
R: Sim! Aumente fragmentsNeeded em GameManager (default 3)

P: Maria Sangrenta fica muito lenta?
R: Aumente chaseSpeed em GhostAI (default 6)
